# Priyanshu Sahu | Portfolio

This is a modern, 3D animated portfolio website built with React, Vite, Three.js, and Framer Motion.

## Features
- **3D Hero Scene**: Interactive 3D elements using React Three Fiber.
- **Modern UI**: Sleek, futuristic design with glassmorphism and neon accents.
- **Responsive**: Fully optimized for mobile and desktop devices.
- **Sections**: About, Projects, Skills, Certifications, and Contact.

## Tech Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Animation**: Framer Motion, Three.js (React Three Fiber)
- **UI Components**: Radix UI, Lucide Icons
- **Build Tool**: Vite

## Getting Started
### Prerequisites
- Node.js (v18 or later)
- npm or bun

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Priyanshups07/Portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production
To create a production build, run:
```bash
npm run build
```
The output will be in the `dist` folder.
